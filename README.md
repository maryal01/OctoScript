# OctoScript
Compiling:
    `make all`

Executing:
    `./toplevel.native`

Running Test Scripts:
    `./testrunner.py negative_tests/* positive_tests/*`
    The positive test files are present in the positive_tests directory.
    The negative test files are present in the negative_tests directory.
